Introduction

HTML code is responsible for making the website itself, but a website with just html code is a plain, black and white page. In order to add color, fonts or in general to control how your site looks like, you need to use the CSS language.
This simple guide to CSS language will take you through the basic concepts so you can modify the style of your website by yourself. If you want to check on your html code, you can always review
My guide to HTML basics.

CSS stands for Cascading Style Sheets.
CSS describes how HTML elements are to be displayed on screen.
CSS saves a lot of work. It can control the layout of multiple web pages all at once.

-3-

Back in the old days, you could add style right in the html code. You can just add style property and into that you can add the style you want.
For example the code <p style= "color: red;"> will make the text in your paragraph red.

You can already imagine the problem that this method creates. You have a messy styling code inside your html code and when you want a change you have to change manually all those style properties.
To solve this problem, CSS language was created. CSS removed the style formatting from the HTML page and it's normally saved in external .css files. With an external stylesheet file, you can change the look of an entire website by changing just one file!
Then this external .css file is linked with your .html file.

To link your .css file to the .html file you need to add a link tag into the head tag of your html code:
<link rel="stylesheet" href="local address to your .css file">

-4-

CSS has its own syntax, but it's quite simple and straightforward.
A CSS rule-set consists of a selector and a declaration block:
The selector points to the HTML element you want to style. It can be any html tag you want to style, h1, p, img and so on.
The declaration block is surrounded by curly braces {} and it contains one or more declarations separated by semicolons (;). Each declaration includes a CSS name and a value, separated by a colon (:).

For Example, If we wanted the color of our text to be green, we would do it like this:
p{color:green;} We can add as many declarations we want as long we separate them with semicolons.
p{color:green; font-size: 32px; background-color: grey;}

-6-

Selectors

CSS selectors are used to "find" (or select) the HTML elements you want to style and we can divide them into five categories:

  • The CSS element Selector
  • The CSS id Selector
  • The CSS class Selector
  • The CSS Universal Selector
  • The CSS Grouping Selector

The CSS element Selector

The element selector selects HTML elements based on their tagname.

Here, all <p> elements on the page will be with a red text color:
p{color:red;}

-7-

The CSS id Selector

The id selector uses the id attribute of an HTML element to select a specific element. The id selector is used to select one unique element! To select an element with a specific id, write a hash (#) character,
followed by the id of the element. Because the id attribute has to be unique, it's recommended to avoid it. You can use classes instead.

The CSS rule below will be applied only to the HTML element with the id="para1":
#para1{font-size:32px}

The CSS class Selector

The class selector selects HTML elements with a specific class attribute. To select elements with a specific class, write a period (.) character, followed by the class name.

In this example all HTML elements with the class="red" will be red:
.red{color:red;}

-8-

The CSS Universal Selector

The universal selector (*) selects all HTML elements on the page. It's recommended to use it for style that are definitely applied to all of your elements, such as margin, fonts etc.

The CSS Grouping Selector

The grouping selector selects all the HTML elements with the same style definitions. Instead of writing the same code multiple times, it's better to group the selectors to minimize the code.
To group selectors, separate each selector with a comma.

In this example we have grouped together couple of selectors which would take the same style:
p,h1{text-align:center;}

-9-

Typography and Visual Rules

Fonts

font-family

The browsers use a default font to show your text but with CSS is possible to change that font. Remember however three important things:

  • The font in use must be installed on the user's computer. Computers have many fonts pre-installed but it's a good idea to use couple of fonts just in case. Especially if your main font is not so common.
  • The default font is Times New Roman and if you don't change it, you don't have to mention it on your css file.
  • The fonts belong in two main font-categories, the Serif font and the Sans-Serif font. The Serif font has calligraphic letters, while the Sans-Serif is more clean and modern
  • When the name of our font is more than one word, then it's good to have the name in quotes.

font-size

You can change the size as you like.By default the size of the letters is 16px and headings are a bit bigger.

-10-

font-weight

Instead of inserting the <b> or <strong> tag all the time in your html document, you can use font-weight:bold; to make an element or many elements to appear bolder.
However you can and should still use the strong tag if there is strong meaning in your sentence, since it is a semantic tag.

font-style

Similar to font-weight and <b>tag, you can use font-style:italic; instead of <em>, to make an element appear in italic style.

p{font-family: georgia, arial,serif; font-size: 20px; }
h2{font-style:italic; font-weight:bold;}

-11-

Text

text-align

You can control how your text will be aligned. The default value is left. It can also be right and center.

text-transform

You can use the value capitalize to capitalize the first letter of each word, uppercase, lowercase to make every letter capital or small letter.

word-spacing

It's not so common but sometimes you might want to give an extra space between the words. You can set your own distance in px(=pixels).

letter-spacing

Similar to word-spacing you can increase the space between the letters.

p{text-align:center;}
h1{text-transform:uppercase;}

-12-

Colors

In css colors can be specified by using a color name like red, blue, yellow and so on. We can also set a color to the background, using the background-color.
In CSS, colors can also be specified using:

  • RGB values: rgb(255, 99, 71)
  • HEX values: #ff6347
  • HSL values: hsl(9, 100%, 64%)
  • RGBA values: rgba(255, 99, 71, 0.5)
  • HSLA values: hsla(9, 100%, 64%, 0.5)

These values give us more control over the contrast, hue, opacity values of our colors.

opacity

Opacity controls the transparency and it accepts a number from 0-1.

body{background-color: light-gey;}
p{color:orange;}

-13-

Borders

Every html element is surrounded by a border, which is invisible by default. With the border property, you can change the style of it.
The border takes 3 values: border's width, style and color.

  • The width determines how thick/thin is the border. You can set it with pixels (px) or you can use the pre-defined values: thin, medium, thick.
  • Style determines the border's style. There are pre-defined values such as: solid, double, dashed, dotted.
  • The color determines the border's color and it can be any color.

It is possible to make a border to look like a circle with the border-radius property. You can use px or % and the higher the value the more rounded it gets.

button{border:1px solid red; border-radius:100%;} That will make the button a perfect circle.

-14-

The Box Model

As we have already learned, each element takes its own space on the page and the content resides inside that space, that box.
There are 4 properties that can manipulate the space that an element will take on the page and also the space that the content will take inside its own box.
We've already examined the border, which changes the border surrounding the element. The rest are:

  • width and height: manipulate the dimensions of the content itself
  • padding: manipulates the amount of space between the content and the border
  • margin: manipulates the amount of space between the border and the rest of the webpage

width and height

There are default settings by all the browsers but we can change them. If we set a size in pixels, it will be the same across all devices. This method is not so flexible with big sizes
because maybe our content looks fine in a laptop, but it will overflow a mobile screen. Another method is to set a size in percentage %. Then the element will keep the same amount of screen-space across all devices.

-15-

padding and margin

padding can take 4 values: padding-top, padding-right, padding-bottom, padding-left controlling each side of the element. However if you want to change all sides
you can set all 4 values together by typing just padding. In this case, the sizes you type work clockwise, meaning the values are: top-left-bottom-right.
If all the values are the same size, then it's enough if you declare just one size. If the values are same in pairs, like some top-bottom and same left-right, then it's enough to declare 2 sizes.

h1{padding-bottom:20px;}: This will add 20 pixels space only on the top of the content.
h1{padding:20px;}: This will add 20 pixels space around the content.
h1{padding:20px 30px;}: This will add 20pixels space on the top and bottom of the content and 30 pixels space around the left and right of the content.

margin has the same syntax with padding.

-16-

Flexbox

Flexbox is an easy, yet powerful method to display multiple elements in a flexible, responsive way. To start you need to declare display:flex on a parent element.
Flex automatically will place the children elements on a row. Then you have a lot of control over the way it's displayed.

  • flex-direction:column sets your elements to be displayed in a column.
  • flex-wrap take 2 values, either wrap or nowrap and it controls whether your elements will change the row, if they don't fit on the screen.
  • align-items positions your elements vertically inside the flexbox. This property is declared on the parent container and its value can be:
    • start: elements are positioned to the top of the container
    • end: elements are positioned to the bottom of the container
    • center
    • stretch

-17-

  • justify-content positions your elements horizontally inside the flexbox. This property is declared on the parent container and its value can be:
  • start: elements are positioned to the left of the container
  • end: elements are positioned to the right of the container
  • center: elements are positioned to the center of the container
  • stretch: elements are stretched across the container
  • space-around: it adds equal space around each element
  • space-between: it adds space only between the elements
  • space-evenly: it adds equal amount of space around and between each element
